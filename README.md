# SM

## description

Strange Machine. <br/>
experiment with creating virtual machine based language in C with own bytecode and assembly

## installation

```console
make install
```

## assembly examples

<a href="./examples">here</a>

## usage

```console
# execute some assembly file
sma run file.sma

# compile assembly to sm bytecode
sma compile file.sma file.smb

# run bytecode
sme file.smb
```
